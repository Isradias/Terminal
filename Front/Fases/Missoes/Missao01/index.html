info INSTRUÇÕES

Você deve navegar pelo sistema de arquivos remoto para encontrar evidências de intrusão. Utilize o comando ls para listar diretórios e cd para acessá-los.

COMANDOS RÁPIDOS

ls

Lista os arquivos e pastas do diretório atual.

cd [nome]

Altera o diretório de trabalho para a pasta especificada.

cat [arquivo]

Exibe o conteúdo textual de um arquivo específico.

clear

Limpa todo o histórico visível do terminal atual.

pwd

Mostra o caminho completo do diretório atual no terminal.
